# Wiki

Wiki is your team's knowledge base for organizing and sharing information.

## Creating Pages

Click the "New Page" button in the sidebar to create a page. The block editor lets you add headings, text, lists, code blocks, and more.

## Spaces

Pages are organized into spaces. Access permissions can be configured per space.

## Revision History

Edit history is saved automatically. You can review past versions and restore them if needed.

## Semantic Search

When Embedding is enabled by an admin, search results include semantically related pages in addition to keyword matches.
