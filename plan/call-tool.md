# 社内通話ツール 仕様書

## 1. 概要

社内通話ツールは、リモートワーク環境における社内コミュニケーションを活性化するための機能です。2つのモードを提供し、それぞれ異なる用途に対応します。

## 2. モード構成

### 2.1 Everybody Coworkingモード

#### コンセプト
- 全社員が同じ「仮想オフィス空間」に集まり、お互いの「顔」が見える状態
- 物理オフィスの「同じ部屋にいる」感覚をオンラインで再現

#### 機能一覧

| 機能 | 説明 |
|------|------|
| リアルタイム顔表示 | 全参加者のWebカメラ映像をタイル表示 |
| 在席状況表示 | オンライン/離席中/オフラインのステータス表示 |
| ミュート状態表示 | 各参加者のミュート状態を視覚化 |
| ワンクリック接続 | ワンクリックでCoworking空間に参加 |
| プレゼンス更新 | 参加者の入退室をリアルタイムに通知 |
| **オフィスカメラ** | 物理オフィスの様子を大きく表示 |

#### 参加者ロール

| ロール | 説明 | 表示サイズ |
|--------|------|-----------|
| **オフィスカメラ** | 会社のオフィスを映す固定カメラ | 大（2x2タイル相当） |
| リモート社員 | 自宅や外出先から参加 | 標準（1x1タイル） |

#### UIイメージ

├─────────────────────────────────────────────────────────────┤

│ ││ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ │
│ └───────────────────────┘ │ 高橋 │ │ 山田 │ │ 伊藤 │ │ 渡辺 │ │
│┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ │
│ │ 斎藤 │ │佐々木│ │ 山本 │ │ 中村 │ │ 😊  │ │ 離席 │ │ 😊  │ │ オフ │ │
│ │ 😊  │ │ 😊  │ │ 😊  │ │ 🔇  │ └─────┘ └─────┘ └─────┘ └─────┘ │

#### 技術仕様

| 項目 | 仕様 |
|------|------|
| 映像コーデック | VP8 / H.264 |
| 解像度 | 320x240 (低帯域) / 640x480 (標準) / 1280x720 (オフィスカメラ) |
| フレームレート | 15fps (省電力) / 30fps (標準) |
| 最大参加者数 | 100名 |
| 遅延 | 500ms以下 |

### 2.2 Roomモード

#### コンセプト
- 必要に応じて作成する会議ルーム
- AIが介在し、議事録を自動生成
- 会議の記録が社内ポータルに蓄積

#### 機能一覧

| 機能 | 説明 |
|------|------|
| ルーム作成 | ルーム名、参加者、開始時刻を設定して作成 |
| 画面共有 | プレゼンテーション、ドキュメント共有 |
| **話者分離** | 誰が発言したかを自動識別 |
| **リアルタイム文字起こし** | 発言内容をリアルタイムでテキスト化 |
| **録画** | 映像・音声をクラウドに保存 |
| **AI要約** | 会議内容の要約を自動生成 |
| **議事録生成** | Wiki形式で議事録を自動作成 |

#### AI機能詳細

**話者分離（Speaker Diarization）**
```
発言者識別フロー:
音声ストリーム → Google Cloud Speech-to-Text → 話者分離
    ↓
話者ID割り当て（Speaker 1, Speaker 2...）
    ↓
ユーザー紐付け（初回のみ手動、以降は自動学習）
```

**リアルタイム文字起こし**
```
仕様:
- 言語: 日本語（自動検出）
- 認識精度: 95%以上を目標
- 遅延: 2秒以内
- 専門用語登録: 可能（社内用語辞書）
```

**録画**
```
保存先: Google Cloud Storage
形式: MP4 (映像+音声)
画質: 1080p (画面共有時) / 720p (顔映像)
保持期間: 無期限
```

**AI要約・議事録生成**
```
処理フロー:
文字起こしテキスト → Vertex AI (Gemini) → 構造化議事録
    ↓
- サマリー（3-5行）
- 決定事項リスト
- アクションアイテム（担当者、期限）
- 次回議題候補
```

#### 議事録のWiki連携

生成された議事録は自動的にWikiページとして作成:

```
/meetings/
  /2024/
    /03/
      /2024-03-15_定例会議/
        - 議事録.md (自動生成)
        - 録画.mp4 (リンク)
        - 文字起こし.txt (全文)
```

## 3. 技術アーキテクチャ

### 3.1 使用技術

| カテゴリ | 技術 | 用途 |
|---------|------|------|
| WebRTC | LiveKit | リアルタイム映像・音声 |
| フロントエンド | @livekit/components-react | Room UI |
| 音声認識 | Google Cloud Speech-to-Text | 文字起こし・話者分離 |
| AI | Vertex AI (Gemini) | 要約・議事録生成 |
| 録画 | LiveKit Egress | クラウド録画 |
| ストレージ | Cloud Storage | 録画・文字起こし保存 |

### 3.2 データフロー

```
会議開始│
    ├──▶ Everybody Coworkingモード
    │        └──▶ 全参加者の映像配信（オフィスカメラは大画面表示）
    │
    └──▶ Roomモード
             ├──▶ 録画開始 (LiveKit Egress)
             ├──▶ 音声 → Speech-to-Text → 文字起こし
             │
             ▼
会議終了
    ├──▶ 録画・文字起こし → Cloud Storage保存
    └──▶ AI要約生成 → Wiki議事録自動作成
```

## 4. 非機能要件

| 項目 | Everybody Coworking | Room |
|------|---------------------|------|
| 最大参加者数 | 100名 | 50名 |
| 映像遅延 | < 500ms | < 300ms |
| 文字起こし遅延 | - | < 2秒 |
| 録画品質 | - | 1080p |
| オフィスカメラ解像度 | 1280x720 | - |

## 4. データベーススキーマ追加

```sql
-- 通話ルーム
CREATE TABLE call_rooms (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    mode TEXT NOT NULL, -- 'coworking', 'meeting'
    creator_id TEXT REFERENCES users(id),
    livekit_room_id TEXT UNIQUE NOT NULL,
    status TEXT DEFAULT 'active', -- 'active', 'ended'
    started_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    ended_at TIMESTAMP,
    recording_url TEXT,
    transcript_url TEXT
);

-- ルーム参加者
CREATE TABLE room_participants (
    id TEXT PRIMARY KEY,
    room_id TEXT REFERENCES call_rooms(id) ON DELETE CASCADE,
    user_id TEXT REFERENCES users(id),
    joined_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    left_at TIMESTAMP,
    speaker_label TEXT -- Speech-to-Textの話者ラベル
);

-- 議事録
CREATE TABLE meeting_minutes (
    id TEXT PRIMARY KEY,
    room_id TEXT REFERENCES call_rooms(id),
    page_id TEXT REFERENCES pages(id), -- Wikiページ紐付け
    summary TEXT,
    decisions JSONB, -- 決定事項リスト
    action_items JSONB, -- アクションアイテム
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 文字起こしセグメント
CREATE TABLE transcript_segments (
    id TEXT PRIMARY KEY,
    room_id TEXT REFERENCES call_rooms(id) ON DELETE CASCADE,
    speaker_label TEXT,
    user_id TEXT REFERENCES users(id),
    content TEXT NOT NULL,
    start_time DECIMAL, -- 秒
    end_time DECIMAL,
    confidence DECIMAL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- インデックス
CREATE INDEX idx_rooms_status ON call_rooms(status);
CREATE INDEX idx_participants_room ON room_participants(room_id);
CREATE INDEX idx_transcripts_room ON transcript_segments(room_id);
```

    ├── POST   /:room_id/regenerate # 議事録再生成
    └── PUT    /:id                 # 議事録編集

### 2.2 Roomモード

#### コンセプト
- 必要に応じて作成する会議ルーム
- AIが介在し、議事録を自動生成
- 会議の記録が社内ポータルに蓄積

#### 機能一覧

| 機能 | 説明 |
|------|------|
| ルーム作成 | ルーム名、参加者、開始時刻を設定して作成 |
| 画面共有 | プレゼンテーション、ドキュメント共有 |
| **話者分離** | 誰が発言したかを自動識別 |
| **リアルタイム文字起こし** | 発言内容をリアルタイムでテキスト化 |
| **録画** | 映像・音声をクラウドに保存 |
| **AI要約** | 会議内容の要約を自動生成 |
| **議事録生成** | Wiki形式で議事録を自動作成 |

#### AI機能詳細

**話者分離（Speaker Diarization）**
```
発言者識別フロー:
音声ストリーム → Google Cloud Speech-to-Text → 話者分離
    ↓
話者ID割り当て（Speaker 1, Speaker 2...）
    ↓
ユーザー紐付け（初回のみ手動、以降は自動学習）
```

**リアルタイム文字起こし**
```
仕様:
- 言語: 日本語（自動検出）
- 認識精度: 95%以上を目標
- 遅延: 2秒以内
- 専門用語登録: 可能（社内用語辞書）
```

**録画**
```
保存先: Google Cloud Storage
形式: MP4 (映像+音声)
画質: 1080p (画面共有時) / 720p (顔映像)
保持期間: 無期限
```

**AI要約・議事録生成**
```
処理フロー:
文字起こしテキスト → Vertex AI (Gemini) → 構造化議事録
    ↓
- サマリー（3-5行）
- 決定事項リスト
- アクションアイテム（担当者、期限）
- 次回議題候補
```

## 4. データベーススキーマ追加

```sql
-- 通話ルーム
CREATE TABLE call_rooms (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    mode TEXT NOT NULL, -- 'coworking', 'meeting'
    creator_id TEXT REFERENCES users(id),
    livekit_room_id TEXT UNIQUE NOT NULL,
    status TEXT DEFAULT 'active', -- 'active', 'ended'
    started_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    ended_at TIMESTAMP,
    recording_url TEXT,
    transcript_url TEXT
);

-- ルーム参加者
CREATE TABLE room_participants (
    id TEXT PRIMARY KEY,
    room_id TEXT REFERENCES call_rooms(id) ON DELETE CASCADE,
    user_id TEXT REFERENCES users(id),
    joined_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    left_at TIMESTAMP,
    speaker_label TEXT -- Speech-to-Textの話者ラベル
);

-- 議事録
CREATE TABLE meeting_minutes (
    id TEXT PRIMARY KEY,
    room_id TEXT REFERENCES call_rooms(id),
    page_id TEXT REFERENCES pages(id), -- Wikiページ紐付け
    summary TEXT,
    decisions JSONB, -- 決定事項リスト
    action_items JSONB, -- アクションアイテム
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 文字起こしセグメント
CREATE TABLE transcript_segments (
    id TEXT PRIMARY KEY,
    room_id TEXT REFERENCES call_rooms(id) ON DELETE CASCADE,
    speaker_label TEXT,
    user_id TEXT REFERENCES users(id),
    content TEXT NOT NULL,
    start_time DECIMAL, -- 秒
    end_time DECIMAL,
    confidence DECIMAL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- インデックス
CREATE INDEX idx_rooms_status ON call_rooms(status);
CREATE INDEX idx_participants_room ON room_participants(room_id);
CREATE INDEX idx_transcripts_room ON transcript_segments(room_id);
```
