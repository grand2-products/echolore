# コスト試算

## 1. 月額ランニングコスト

### 1.1 Cloudflare料金

| サービス | プラン | 月額 | 備考 |
|---------|--------|------|------|
| **Pages** | Free | $0 | 無料枠で十分 |
| **Workers** | Paid | $5/月 | 10M リクエスト含む |
| **D1** | Paid | $5/月 | 5GBストレージ + 読み書き |
| **R2** | 従量 | ~$1/月 | 10GB想定 |
| **Calls** | Free → Standard | $0-$20 | 無料枠: 1GB/月 |

**月額合計: $11-31/月** (約 ¥1,600-4,600)

### 1.2 詳細内訳

#### Workers

```
無料枠 (Free プラン):
- 100,000 リクエスト/日

Paid プラン ($5/月):
- 10,000,000 リクエスト/月含む
- 超過: $0.50 / 1M リクエスト

想定使用量 (社員50人):
- 1人/日 平均 50 APIリクエスト
- 50人 × 50req × 22日 = 55,000 req/月
→ 無料枠で十分だが、Paid必要な機能のため $5/月
```

#### D1 (SQLite)

```
無料枠:
- 5GB ストレージ
- 500万 行読み取り/日
- 10万 行書き込み/日

Paid ($5/月):
- 25GB ストレージ
- 2500万 行読み取り/日
- 50万 行書き込み/日

想定使用量:
- ページ数: 1,000ページ
- ブロック数: 10,000ブロック
- データサイズ: ~100MB
→ 無料枠で十分だが、本番はPaid推奨
```

#### R2 (Storage)

```
料金:
- ストレージ: $0.015/GB/月
- Class A操作: $4.50/100万回
- Class B操作: $0.36/100万回
- Egress: 無料 ← 重要！

想定使用量:
- 画像・ファイル: 10GB
- ストレージ: 10GB × $0.015 = $0.15/月
- 操作: 数ドル以内
→ 合計 $1/月 程度
```

#### Cloudflare Calls

```
Free プラン:
- 1GB 転送/月
- 最大10参加者/ルーム

Standard ($5/月):
- 無料枠 + 従量課金
- 超過: $0.10/GB

想定使用量:
- 全社ミーティング: 週1回 × 1時間
- 50人参加 × 1Mbps = 50Mbps
- 1時間 = 22.5GB
- 月4回 = 90GB

→ 無料枠超過
→ Standard プラン必要: ~$20/月
```

## 4. コスト最適化戦略

### 4.1 短期 (リリース時)

- Calls: 無料枠内で運用 (会議時間制限)
- Workers: Free プラン検討
- R2: 不要ファイル削除

### 4.2 中期 (50人以上)

- Calls: Standard プラン移行
- D1: 有料プラン検討 (バックアップ含む)
- Cloudflare Analytics 導入

### 4.3 長期 (100人以上)

- マルチテナント対応
- Cloudflare Enterprise 検討
- 別リージョン展開

## 5. 注意点

### 5.1 隠れコスト

| 項目 | リスク | 対策 |
|------|--------|------|
| Calls超過 | 会議増加で課金増 | 月次監視 |
| D1容量 | Wiki増加で容量圧迫 | 定期アーカイブ |
| R2容量 | 画像増加 | サイズ制限 |

### 5.2 コスト監視

```bash
# Cloudflare Dashboard
Workers → Analytics → Requests/CPU time
D1 → Database → Storage/Read/Write
R2 → Bucket → Storage/Operations
Calls → Analytics → Minutes/GB
```

**アラート設定推奨:**
- Workers: 5M req/月 超過時
- D1: 3GB 超過時
- Calls: $15/月 超過時
