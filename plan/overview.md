# 社内ポータルアプリ 企画概要

## 1. 背景・目的

### 背景
- リモートワーク・ハイブリッド勤務の増加により、社内コミュニケーションの課題が顕在化
- 情報がSlack、Notion、Google Driveなどに散在し、ナレッジの検索性が低下
- 全社員参加のミーティングにおいて、手軽なビデオ会議ツールが不足

### 目的
1. **情報の一元管理**: 社内Wikiでナレッジを集約
2. **認証の簡素化**: GoogleアカウントでのSSO実現
3. **コミュニケーション活性化**: 簡易ビデオ会議機能の提供

## 2. ターゲットユーザー

- grand2 Products 全社員
- 想定規模: 10-100名（スケーラブル設計）

## 3. 主要機能

### 3.1 認証・認可
- Google SSO (Cloudflare Access)
- 社内ドメイン制限 (@grand2-products.com)
- ロールベースアクセス制御

### 3.2 社内Wiki
- NotionライクなBlock-basedエディタ
- リアルタイム共同編集
- ページ階層構造
- フルテキスト検索
- 画像・ファイル添付 (R2 Storage)

### 3.3 ビデオ会議
- WebRTCベース（Cloudflare Calls）
- 全社員参加型ミーティング
- ミーティングルーム作成・管理
- 画面共有

## 4. 非機能要件

| 項目 | 要件 |
|------|------|
| パフォーマンス | ページ読み込み < 2秒 |
| 可用性 | 99.9%以上 |
| 同時接続数 | 100名対応 |
| データ保持 | 無期限（DBバックアップあり） |
| セキュリティ | HTTPS必須、SSO認証 |

## 5. 成功指標 (KPI)

1. Wiki利用率: 月間アクティブユーザー 80%以上
2. ビデオ会議利用: 週1回以上の全社ミーティング実施
3. 検索成功率: 90%以上

## 6. スコープ外

- モバイルアプリ（Web対応のみ）
- 外部ユーザー招待機能
- 高度な権限管理（部署単位など）
- AI機能（将来検討）
